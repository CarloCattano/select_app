# select_app
rofi wayland app chooser

```bash
rofi -V
Version: 1.7.5+wayland1-41-g61f96e4c (wayland)
```

example bind:
```basg 
bind=$Mod, G, exec, rofi -theme ~/.config/PATH/TO/THEME/apps.rasi -i -I -show-icons -show window -selected-row 0 -p "Select app"
```
